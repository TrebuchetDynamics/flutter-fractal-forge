# assets

## Purpose
Static assets bundled with the application. Contains the app icon and 250+ catalog thumbnail images used in the fractal catalog browser.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `icon/` | App launcher icon (`ic_launcher.png`) |
| `catalog_thumbs/` | 250+ fractal thumbnail PNG images for the catalog grid |

## Key Files

| File | Description |
|------|-------------|
| `feature-graphic.png` | Play Store feature graphic |

## For AI Agents

### Working In This Directory
- Catalog thumbnails are generated by `integration_test/generate_gpu_thumbnails_test.dart`
- Thumbnails follow naming convention: `{fractal_id}.png` (e.g., `mandelbrot.png`)
- Assets are declared in `pubspec.yaml` under `flutter.assets`
- Do NOT manually edit generated thumbnails; regenerate via integration test
