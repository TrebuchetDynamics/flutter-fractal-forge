KEY FINDINGS
================================================================================

1. Smooth Iteration Count is Essential
   └─ Single GLSL line: iter + 1 - log(|z|)/log(2)
   └─ Dramatic visual improvement, no performance cost
   └─ All projects use this (DeepDrill, FractalExplorer, Fractals-Explorer)

2. Perturbation Theory Enables Unlimited Zoom
   └─ 50-95% speedup via reference orbit + delta computation
   └─ Required for deep zoom (beyond 2^50x)
   └─ Complex but powerful (DeepDrill example)

3. GPU Shaders Are the Sweet Spot
   └─ Better than pure CPU for interactive performance
   └─ More portable than pure OpenCL
   └─ All modern projects use GLSL (FractalExplorer, Fractals-Explorer)

4. Multi-Algorithm Coloring Matters
   └─ 6+ proven algorithms in Fractals-Explorer
   └─ Real-time switching without re-rendering
   └─ Low performance cost, high user satisfaction

5. Newton Fractals Are Worth Implementing
   └─ Mathematically distinct from escape-time
   └─ 50 lines of GLSL
   └─ Visually striking results (FractalExplorer, Fractals-Explorer)
