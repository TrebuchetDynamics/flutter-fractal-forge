hashtag the app already stamps on every share.

## The growth loop (now wired into the app)

```
post a fractal → people download (free, no ads) → they explore → tap Share
   → caption auto-carries @FractalForge + #fractalforge + a tappable link
   that reopens the EXACT view → others tap it, land on that fractal, riff,
   share their own → you reshare the best → loop tightens
```

In-app mechanics supporting this (live as of this change):
- **Share caption**: every shared image/GIF reads
  `"{Fractal} — made free with Fractal Forge / Open this exact view 👉 {link} / @FractalForge #fractalforge"`.
- **Reproducible deep link**: the link is a universal URL
  (`https://fractal.trebuchetdynamics.com/view?...`) that reopens the exact
  module + camera + rotation + params — a one-tap "coordinate drop".
- **Watermark toggle** (in Export → Advanced, **off by default**): stamps
  `@FractalForge` in the corner so reshared screenshots stay attributed.

## What success looks like

| Horizon | Target |
|---|---|
| Weeks 1–6 | `#fractalforge` is no longer empty; first organic user shares; 2–5k downloads. |
| Months 2–6 | recognized handle in generative-art/math-viz circles; recurring UGC; reshares from larger accounts; 1–2 open-source contributors. |
| 6–12 months | "the free fractal app"; community-driven catalog growth; beta/feedback pipeline; GitHub stars; Trebuchet credibility halo. |

Because the app is free + open-source + no ads, the "conversion" is **reach,
reputation, contributors, and studio credibility** — not revenue.
