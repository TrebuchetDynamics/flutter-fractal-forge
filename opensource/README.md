# Open-source fractal references

This folder stores third-party fractal reference projects and the local analysis distilled from them.

- `docs/` — catalogs, implementation notes, and refactor roadmap derived from the reference projects.
- `scripts/` — helper scripts for cloning/updating reference projects.
- `repos/` — clean checked-in gitlink references to upstream projects.
- `par-fractal/` — temporarily left at the root because its nested working tree has local modifications.
