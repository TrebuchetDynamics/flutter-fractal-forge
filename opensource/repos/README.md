# Open-source reference repositories

Vendored upstream fractal/reference projects are grouped by how this Flutter app uses them:

- `renderers/` — complete fractal renderer applications and engines used for implementation techniques, precision strategies, and UI/rendering reference.
- `formula-catalogs/` — large formula, shader, attractor, and variation corpora used for catalog and shader research.
- `research-toolkits/` — broader research/toolkit repositories that are useful references but not directly fractal-renderer apps.

These directories contain upstream repositories and should be treated as reference inputs rather than app source code. Keep source/license context in tracked notes before adapting ideas into the Flutter app.
