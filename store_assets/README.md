# Play Store Assets

This directory contains assets for Google Play Store and other app stores.

## Required Assets

### Feature Graphic
- **Size**: 1024 x 500 pixels
- **Format**: PNG or JPEG
- **Purpose**: Top banner on Play Store listing
- **File**: `feature_graphic.png`

### App Icon (Play Store)
- **Size**: 512 x 512 pixels
- **Format**: PNG (32-bit PNG with alpha)
- **Purpose**: App listing icon on Play Store
- **File**: `play_store_icon.png`

### Phone Screenshots
- **Count**: 2-8 phone screenshots
- **Format**: PNG or JPEG
- **Resolution**: 16:9 or 9:16 aspect ratio
- **Max size**: 8 MB each
- **Directory**: `screenshots/phone/`

### Tablet Screenshots (Optional)
- **Count**: 2-8 tablet screenshots
- **Format**: PNG or JPEG
- **Resolution**: 16:9 or 9:16 aspect ratio
- **Directory**: `screenshots/tablet/`
